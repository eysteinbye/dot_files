My OS X dot.files
=================

*Work in progress*

This is the dot.files I use on my Mac. Might not be the right for you :)

## Prerequisites

1. Install iTerm
2. HomeBrew
3. Install rbenv
4. Install Bundler
5. brew install ctags-exuberant
6. Oh-my-Zsh

## Installation

$ git clone git@github.com:eysteinbye/dot_files.git ~/.dot_files && ~/.dot_files/setup.sh
